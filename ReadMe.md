### Teste Arquitetura

Boilerplate desenvolvido com base nas especificações do teste; build com `webpack`.


## Configuração

- `npm install`
- Build para dev - `npm run build-dev`
- Build para prod - `npm run build-prod`
- Testes - `npm run test`
- Abre em localhost:3000 (auto watch) - `npm start`

## Incluso

- `webpack`
- `babel`
- `eslint`
- `axios`
- `Styled Components`
